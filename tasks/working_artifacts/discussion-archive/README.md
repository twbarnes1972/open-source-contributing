# Discussion Archive

Dated offload artifacts from [DISCUSSIONS.md](../../../DISCUSSIONS.md) `## Recently concluded`
sweeps — one file per sweep: `archive-YYYY-MM-DD.md`.

Each artifact holds the concluded engagement rows verbatim plus optional per-engagement
observations. They are the bounded, **disposable** index of concluded upstream engagements;
the durable record of each engagement is its owning task's `## Work Completed`, and git
history is the deep backup. See
[tasks/discussion_handling.md](../../discussion_handling.md) for the lifecycle.
